# 딥러닝 실습 01: 폐암 수술 환자 생존 예측 (이진 분류)

**파일**: `260107 딥러닝 01.py`
**데이터**: `ThoraricSurgery3.csv` (폐암 수술 환자 데이터)

## 1. 핵심 내용

이 코드는 가장 기본적인 **이진 분류(Binary Classification)** 딥러닝 모델을 구현합니다.
수술 전 진료 기록(16개 항목)을 보고, 수술 1년 후 생존(0) 또는 사망(1)을 예측합니다.

## 2. 주요 코드 포인트

### (1) 데이터 로드 (Numpy)

```python
# 구분자(comma)로 된 CSV 파일을 수치 행렬로 불러옴
Data_set = np.loadtxt(file_path, delimiter=",")
```

- 여기서는 `pandas` 대신 `numpy`를 사용하여 숫자 데이터를 직접 로드했습니다.

### (2) 데이터 전처리 (Slicing)

```python
X = Data_set[:, 0:16]  # 속성 (환자 정보)
y = Data_set[:, 16]    # 타겟 (생존 여부)
```

- 전체 데이터셋에서 `X`(입력)와 `y`(정답)를 인덱싱으로 분리했습니다.

### (3) 모델 구조 (Sequential)

```python
model = Sequential()
model.add(Input(shape=(16,)))        # 입력층 (속성 16개)
model.add(Dense(30, activation='relu')) # 은닉층
model.add(Dense(1, activation='sigmoid')) # 출력층
```

- **입력층**: 속성이 16개이므로 `shape=(16,)`
- **출력층**: 생존/사망 두 가지 중 하나를 확률로 예측하므로 **뉴런 1개**, 활성화 함수는 **`sigmoid`** 사용.

### (4) 학습 설정 (Compile)

```python
model.compile(loss='binary_crossentropy', optimizer='adam', metrics=['accuracy'])
```

- **오차 함수**: 이진 분류이므로 `binary_crossentropy` 필수.

## 3. 요약

- **목표**: O/X 문제 (이진 분류) 해결
- **특징**: 가장 기초적인 딥러닝 구조. 훈련 데이터와 평가 데이터의 분리 없이 전체 데이터를 학습 및 평가에 사용함 (초급 단계).
