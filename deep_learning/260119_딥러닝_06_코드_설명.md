# 주택 가격 예측 AI 프로세스 (`260108 딥러닝 06.py`)

이 문서는 딥러닝을 처음 접하는 분들을 위해 `06.py` 파일의 주택 가격 예측 프로세스를 **요리 과정**에 비유하여 쉽고 자세하게 설명합니다.

---

## 1. 재료 준비: 데이터 로드 (Data Load)

가장 먼저 요리에 필요한 재료를 준비하듯, 주택 정보가 담긴 데이터 파일(`house_train.csv`)을 불러옵니다. 이 데이터에는 집의 크기, 방 개수, 지하실 면적 등 다양한 정보와 실제 집값이 들어있습니다.

```python
df = pd.read_csv(file_path) # CSV 파일(재료)을 불러옵니다.
```

## 2. 재료 손질: 데이터 전처리 (Preprocessing)

재료를 그대로 쓸 수 없으니 다듬는 과정입니다.

- **결측치 채우기 (`fillna`)**: 정보가 비어있는 칸이 있으면 통계적으로 무난한 **'평균값'**으로 채워 넣습니다.
- **숫자로 변환하기 (`get_dummies`)**: 컴퓨터는 "TV 있음" 같은 글자를 이해하지 못합니다. 이를 **숫자(1, 0)**로 바꿔주는 작업을 '원-핫 인코딩'이라고 합니다.

## 3. 핵심 재료 선정: 상관관계 분석 (Correlation Analysis)

모든 재료를 다 넣는다고 맛있는 요리가 되진 않죠. 집값에 진짜 영향을 주는 중요한 요소만 골라냅니다.

- **상관관계 확인 (`corr`)**: 각 요소들이 집값과 얼마나 관련이 있는지 계산합니다.
- **속성 선택**: 그래프를 그려보니 **'집 품질(OverallQual)', '생활 면적(GrLivArea)', '차고 크기'** 등이 집값과 아주 밀접하다는 걸 발견하고, 이것들만 학습에 사용하기로 결정합니다.

## 4. 재료 나누기: 학습/테스트 데이터 분리

가지고 있는 문제집을 몽땅 공부하는 데 쓰면 나중에 시험 칠 문제가 없겠죠?

- **학습용(Train Set, 80%)**: AI가 공부할 때 쓰는 문제집입니다.
- **평가용(Test Set, 20%)**: 나중에 실력을 확인하기 위해 남겨두는 시험지입니다.

## 5. AI 뇌 설계: 모델 구조 만들기 (Model Design)

이제 AI의 두뇌를 만듭니다. `Sequential`이라는 빈 틀에 뉴런 층(`Dense`)을 층층이 쌓습니다.

1.  **입력층**: "이 집은 30평이고, 방이 3개야" 같은 정보가 들어옵니다.
2.  **은닉층**: 들어온 정보를 가지고 복잡한 계산을 수행하며 특징을 파악합니다.
3.  **출력층**: 최종적으로 **"이 집은 5억 원이야!"** 하고 숫자 하나를 뱉어냅니다. (회귀 모델이므로 활성화 함수 없이 숫자 그대로 출력)

## 6. 요리(학습) 시작: 모델 훈련 (Training)

AI에게 학습 데이터를 주고 훈련을 시킵니다.

- **`fit()`**: AI가 정답을 맞혀보는 과정입니다. 틀린 만큼 오차를 줄여가며 점점 똑똑해집니다.
- **과적합 방지 (`EarlyStopping`)**: 더 이상 실력이 늘지 않으면("이제 다 배운 것 같은데?") 시간 낭비하지 말고 **공부를 멈추라**고 설정합니다.
- **최고 기록 저장 (`ModelCheckpoint`)**: 공부하다가 성적이 제일 좋았던 순간을 **자동으로 저장**해 둡니다.

## 7. 맛보기: 결과 예측 및 시각화 (Visualization)

마지막으로, 아까 남겨둔 **시험지(테스트 데이터)**를 줍니다.

- **예측**: AI가 예측한 집값과 실제 집값을 비교해 봅니다.
- **그래프 확인**: 파란 선(AI 예측값)과 주황 선(실제 값)이 비슷하게 겹쳐진다면, 아주 훌륭한 AI가 탄생한 것입니다!
